Vraag: Waarom staat de tekst in de blokken horizontaal gecentreerd?
- omdat er text-align:center wordt toegepast in de div elementen.

Vraag: Ga na dat een property vertical-align:center geen effect heeft. Waarom eigenlijk niet?
- omdat vertical-align alleen is om een inline element block in verticaal te plaatsen in een andere block, bv een foto in een lijntje text.

Vraag: Ga na dat vertical-align: middle er niet voor zorgt dat de tekst verticaal gecentreerd wordt. Wat doet vertical-align dan wel?
- Vertical-align zorgt ervoor dat je bv een image in een block een plaats kan geven.

Vraag: Wat is de ‘simpele’ manier om een één enkele regel tekst verticaal te centreren?
- line height:normal; is een manier om verticaal te centreren.

Vraag: Welke truc wordt vaak gebruikt om meerdere regels verticaal te centreren?
- display:table cell; en vertical-align:middle, zorgen ervoor dat je meerdere regels kan centreren.

Vraag: Waarom krijgt de regel voor class 'tall' voorrang?
- Omdat die een grotere css waarde heeft.
