Antwoorden opdracht 02

  • Waar wordt de event listener gekoppeld aan de slider?
    - Regel 3 in de JS code, let sliders = document.getElementsByClassName("slider");

  • Waarom moeten we die op twee soorten events koppelen?
    - Omdat als je de slider verschuift wordt de waarde telkens verandert, de 2de event zorgt ervoor dat de waarde wordt geupdate naar de nieuwe waarde dat je hebt geselecteerd op de slider

  • In de CSS file wordt nergens een rode kleur opgegeven, waar wordt dan wel de rode kleur van het blokje ingesteld?
    - Regel 11 in de JS code, colorDemos[0].style.backgroundColor="red";

  • Waarom schrijven we telkens sliders[0] en colorDemos[0] en niet gewoon sliders of colorDemos?
    - Omdat de slider eigenlijk 1 lange array is van de waarden 0-100 .
